## How to download each set
1. Go to `https://www.sportscardspro.com` and make sure you're logged in (top-right shows your account, not "Login").
2. Search for the set name (e.g. "2025 Topps Chrome Platinum Baseball") or go directly to the set's price-guide page.
3. Confirm you're on the **set's price guide page** (a big table of every card + prices), NOT a single card page and NOT search results.
4. Scroll to the **bottom of the price table**. There is a **"Download Price Guide" / "Download Price List"** link/button (CSV). Click it.
5. The CSV downloads to the Downloads folder. Confirm the file appeared.
6. Report `SET NAME → downloaded ✓` or `SET NAME → NOT FOUND / no download button` and move on.

If a set page doesn't exist on SCP or has no download button, skip it and note it. Don't get stuck — keep moving down the list.
